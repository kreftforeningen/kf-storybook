# kf-tailwind

Package to be used together with the main Tailwind CSS package.

- [NPM.js](https://www.npmjs.com/package/kf-tailwind)
- [GitHub](https://github.com/kreftforeningen/kf-tailwind)

## Install

```
pnpm add kf-tailwind
```

## Usage

Import it after Tailwind, like this:

```
@import "tailwindcss";
@import "kf-tailwind/index.css";
```
